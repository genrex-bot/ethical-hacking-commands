## 🚀 Getting Started

If you're brand new to ethical hacking, here's the recommended order:

1. Start with **Networking** — understand how networks work
2. Move to **Host Discovery** — find what's on a network
3. Learn **Scanning & Mapping** — understand what services are running
4. Study **Routing** — understand how traffic flows
5. Explore **Web Recon** — learn web application basics
6. Practice **Password Attacks** in a safe lab environment
7. Try **Exploitation** only in legal lab environments (e.g., HackTheBox, TryHackMe)

## 🧰 Recommended Beginner Tools

| Tool | Purpose | Install |
|------|---------|---------|
| `nmap` | Network scanning | Pre-installed on Kali Linux |
| `netcat (nc)` | Network utility | Pre-installed on most Linux |
| `whois` | Domain lookup | `sudo apt install whois` |
| `dig` | DNS lookup | Pre-installed on most Linux |
| `wireshark` | Packet analysis | `sudo apt install wireshark` |
| `metasploit` | Exploitation framework | Pre-installed on Kali Linux |
| `hydra` | Password brute force | Pre-installed on Kali Linux |
| `aircrack-ng` | Wireless analysis | Pre-installed on Kali Linux |
